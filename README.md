# Dog & Partridge Sheffield ☘️

The official website for Dog & Partridge, an Irish Pub in Sheffield city centre.

## About

Irish owned, cosy, dog friendly & Irish Free House in Sheffield city centre. Home of the Holy Trinity of Beamish, Guinness & Murphy's.

**Accolades:**
- Member of the inaugural Guinness Harp Guide 2026
- Sunday Times Top 10 Pubs in Sheffield
- Exposed Awards Best Traditional Pub 2024 & 2025
- Best Guinness in Sheffield - 10/10 (Girl Drinks Guinness)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` - Home page with hero, accolades, opening hours, and key information
- `/menu` - Full text menu with Irish names and prices
- `/find-us` - Address, contact information, and opening hours

## Build

```bash
npm run build
```
